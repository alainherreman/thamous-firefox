# Extension Firefox Thamous

## Téléchargement

Téléchargez le fichier :

- `thamous-firefox-extension-0.2.0-signed.xpi`

## Installation dans Firefox

1. ouvrir `about:addons`
2. cliquer sur la roue dentée
3. choisir **Installer un module depuis un fichier…**
4. sélectionner le fichier `.xpi`

## À quoi sert cette extension ?

Elle permet d’importer plus facilement dans Thamous :

- la référence principale d’une page web ;
- toutes les références trouvées dans une page ;
- ou la page elle-même.
