# Extension Firefox Thamous

## Téléchargement

Télécharger ce fichier :

- `thamous-firefox-extension-0.2.3-signed.xpi`

## Installation dans Firefox

1. ouvrir `about:addons`
2. cliquer sur la roue dentée
3. choisir **Installer un module depuis un fichier…**
4. sélectionner le fichier `.xpi` téléchargé

## À quoi sert l’extension ?

Elle permet, depuis Firefox, d’importer plus facilement dans Thamous :

- la référence principale d’une page ;
- toutes les références trouvées dans une page ;
- ou la page elle-même.

## Important

Pour fonctionner correctement, l’extension a besoin qu’un modèle LLM ait été sélectionné au préalable dans l’extension elle-même.
